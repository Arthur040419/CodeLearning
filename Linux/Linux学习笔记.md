## Linux的目录结构

与常用的Windows系统比较，Linux的目录结构有下面几点不同：

1.Windows系统有多个盘符，也就是多个顶级目录，而Linux没有盘符的概念，仅仅只用一个`/`符号就表示根目录，只有一个顶级目录。

![image-20250811230105949](./pictures/image-20250811230105949.png)



2.Linux系统用`/`来作为路径分隔符，而Windows系统一般用`\`来作为路径分隔符（好像`/`和`\`都可以）。





## Linux命令

### Linux命令基础语法

在Linux操作系统中，不管什么命令，都有一个基本的语法格式，如下：

```sh
command [-options] [parameter]
```

command代表命令本身、options代表可选项、parameter代表参数

例如下面这个命令：

```sh
ls -l /opt/var/logs
```

表示以列表的形式显示/opt/var/logs路径下的所有文件，其中ls就是命令、-l就是可选项、/opt/var/logs 就是命令的参数





### Home目录和工作目录

不管是Linux还是Windows系统，它们都是多用户系统，因此每一个用户都有一个专属于自己的目录，这个目录就是Home目录。

在Linux系统中，Home目录一般在`/Home/用户名`下，这个目录就是Home目录。

而工作目录就是当前终端控制台所在的目录，有些终端在启动时会默认把Home目录作为工作目录。



### 特殊路径符

Linux系统中有下面三种特殊路径符：

1、`.`表示当前目录

2、`..`表示上一级目录，如果想表示上一级的上一级，可以用`../..`，上一级的上一级的上一级可以用`../../..`

3、`~`表示Home目录。



### 普通用户与root用户之间的转换

在普通用户身份下，输入以下命令可以转换成root身份：

```sh
su - root
```

然后输入`exit`就可以退回原来的用户。

### rm命令

rm命令取自英文remove，表示删除文件或文件夹。

### find命令

通过find命令，可以查找符合条件的文件，有两种查找方式：1.根据文件名来查找  2.根据文件大小查找

#### 根据文件名查找

find命令的语法如下：


#### 根据文件大小查找

根据文件大小查找文件的语法如下：

```sh
find 起始路径 -size +|- n[KMG]
```

其中：

`+`表示大于、`-`表示小于

`n`代表文件大小的数字

`KMG`表示文件大小单位，K表示kb，M表示mb，G表示gb



示例如下：

查找根目录下文件大小大于100kb的文件


### sudo命令

由于root用户拥有最高权限，可以执行任何命令，因此如果执行了错误的命令很可能会对系统造成不可逆的影响，所以不建议长期使用root用户。

但是使用普通用户时，执行命令可能会受到权限限制，那怎么办呢?

`sudo`命令可以为普通用户提供临时root权限，在要执行的命令最前面加上`sudo`就可以临时以最高权限执行该命令。

如下图所示：

在arthur用户下执行`mkdir`命令时，在前面加上了`sudo`才执行成功。

![image-20250821231251970](./pictures/image-20250821231251970.png)





### sudo认证

并不是所有的用户都能够使用`sudo`命令的，如果所有用户都能使用`sudo`，那还不如一直保持在root用户下。

普通用户在使用`sudo`命令前需要进行sudo认证。

有两种方式进行sudo认证，都需要在root用户下：

1.使用`visudo`命令，会自动进入`/etc/sudoers`文件的vim编辑界面。

2.直接使用vim编辑器打开`/etc/sudoers`文件、

然后在文件末尾添加下面的内容
```sh
arthur ALL=(ALL) NOPASSWD:ALL
用户名            执行sudo时不需要密码
```

![image-20250821232332495](./pictures/image-20250821232332495.png)

此时再次使用arthur用户就可以直接使用`sudo`命令：

## 修改文件/文件夹权限（chmod命令）

`chmod`命令可以用来修改文件/文件夹的权限，该命令只能由root用户执行。

语法如下：

```sh
chmod [-R] 权限 文件/文件夹路径
```

其中`-R`选项表示对文件夹内部的所有文件同样修改相同权限。

使用示例如下：


## 控制服务的启动与关闭（systemctl命令）

在Linux中，很多软件都可以使用`systemctl`命令来控制其启动或停止，这些能被`systemctl`命令控制的软件被称为服务。

## 文件的解压缩

Linux中常用的压缩包格式有两种：`.tar`，`.gz`

`.tar`包并没有对文件起到压缩文件的效果，它仅仅是起到将多个文件归档到一块，是一个封装的效果，压缩成`.tar`不会使文件大小变小

`.gz`才是对文件起到了压缩的效果，它可以使文件体积减小。



### 压缩文件

解压缩文件可以使用`tar`命令，其选项如下：

![image-20250824145055029](./pictures/image-20250824145055029.png)

这么多选项了解一下即可，具体使用一般用下面两个固定的命令：

1.压缩成`.tar`文件

```sh
tar -cvf test.tar 要压缩的文件路径列表
```



2.压缩成`.gz`文件

```sh
tar -zcvf test.tar.gz 要压缩的文件路径列表
```



### 解压文件

解压文件也有几个比较常用的命令组合：

1.将文件解压到当前目录

```sh
tar -xvf test.tar
```



2.将文件解压到指定目录

```sh
tar -xvf test.tar -C 指定目录
```



3.以GZip模式来解压，并将结果存放到指定目录

```sh
tar -zxvf test.tar -C 指定目录
```







### zip格式文件的解压缩

虽然Linux通常不用zip格式，但总是需要用到，所以对于zip格式的文件需要使用`zip`命令来压缩，使用`unzip`命令来解压。



#### zip命令

`zip`命令用来将文件压缩成`.zip`压缩包，其语法如下：

```sh
zip [-r] 参数1 参数2 参数3 。。。参数N
```

`-r`选项表示如果被压缩的文件中带有文件夹就需要带上`-r`选项。

**注意：**第一个参数表示将文件压缩到的路径，不是被压缩文件的路径



#### unzip命令

`unzip`命令用来解压`.zip`文件，语法如下：

```sh
unzip [-d] 参数
```

`-d`选项用来指定压缩结果应该放到哪个路径下

参数是被解压的文件的路径。

比如：

1.将test.zip解压到当前目录

```sh
unzip test.zip
```

2.将test.zip解压到指定目录

```sh
unzip -d /root/test test.zip
```
